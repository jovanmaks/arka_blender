## SetAutor ##

```python
fpdf.set_autor(autor: str, isUTF8 = False)
```
### Description ###

Defines the author of the document.

### Parameters ###

author:
> The name of the author.

isUTF8:
> Indicates if the string is encoded in ISO-8859-1 (False) or UTF-8 (True).
> Default value: False.
