## SetTitle ##

```python
fpdf.set_title(title: str, isUTF8 = False)
```
### Description ###

Defines the title of the document.

### Parameters ###

title:
> The title.

isUTF8:
> Indicates if the string is encoded in ISO-8859-1 (False) or UTF-8 (True).
> Default value: False.
