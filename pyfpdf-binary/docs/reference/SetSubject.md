## SetSubject ##

```python
fpdf.set_subject(subject: str, isUTF8 = False)
```
### Description ###

Defines the subject of the document.

### Parameters ###

subject:
> The subject.

isUTF8:
> Indicates if the string is encoded in ISO-8859-1 (False) or UTF-8 (True).
> Default value: False.
